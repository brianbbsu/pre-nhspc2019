資訊學科能力競賽決賽 模擬賽
12/8（日）上午 9:00 至 下午 2:00
立即報名 競賽資訊
宗旨

為使「108學年度普通型高級中等學校資訊學科能力競賽決賽」達成其目的，即「加強輔導中學生資訊學科教育，提高學生對資訊問題研究的興趣，激發其思考與創造能力，藉以鼓勵學生與校際間的觀摩，提昇資訊教育品質。」，本模擬賽旨在穩定競賽選手對於競賽的表現，並有策略性的面對一場長達五小時的競賽，使各地好手能以真正的實力切磋、交流。

競賽日程
  • 即日起開放報名
  • 12/02（一）晚上 11:59 報名截止
  • 12/05（四）晚上前寄送比賽通知
  • 12/08（日）上午 9:00 至 下午 2:00 舉行
競賽規則

此次模擬賽規則將會盡量符合「108學年度普通型高級中等學校資訊學科能力競賽決賽」之規則，其中重點規則如下：

  • 報名資格為目前就讀高中以下，不論有無參加全國賽者皆可報名。
  • 競賽採電腦自動評測，程式必須依規定上傳至評分主機。
  • 程式可重複上傳及評分，但同一題兩次上傳之間必須間隔二分鐘以上。每題最終分數以該題所有單次評分結果之最高分計算。
  • 程式執行時，每組測試資料執行時間個別計時（以評分主機執行時間為準）。執行時間限制如個別題目所示。程式執行超過執行時間視同未完成，該組測試資料得分將以 0 分計算。
  • 本次模擬賽採線上進行，參賽者不得在比賽進行中使用任何方式查詢與題目相關之資訊，或是使用預先準備好的電子資料。
技術細節
  • 評測系統：CMS (Contest Management System) Version 1.4
  • 允許使用的語言是 C 11、C++ 14 以及 Java。
  • Server 作業系統：18.04.1 LTS (Bionic Beaver)
  • 編譯器版本：gcc 7.4.0-1ubuntu1~18.04.1, g++ 7.4.0-1ubuntu1~18.04.1, javac 1.8.0_222
  • 以 Java 程式語言撰寫程式者，請務必將 public class 以該題英文題目名稱命名。若使用其他 class 名稱導致程式無法自動編譯或執行，將以 0 分計算。
